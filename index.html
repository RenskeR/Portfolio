Wie ben ik?

Hallo, ik ben Renske Rothkranz, een 23 jarige CMD student en ik zit in mijn 2e jaar. Ik ben begonnen een paar jaar geleden met een Mbo opleiding gericht op programmeren, deze had ik gekozen omdat ik toen niet wist wat ik wilde doen. Deze opleiding was niet helemaal mijn ding, maar ik ontdekte hier dat ik programmeren binnen HTML en CSS wel leuk vond. Hierna ben ik door gegaan naar AD-ICT in Heerlen, na 1 Â½ jaar had ik geen motivatie meer voor de opleiding want het paste gewoon niet bij mij. Maar een van mijn docenten had vanuit hier de opleiding CMD aangeraden. In het volgend studiejaar ben ik met deze opleiding begonnen en tot nu toe vind ik dat deze opleiding goed bij mij past.

Opdracht CBS | EGD

Dit project is gemaakt om de European Green Deal in het licht te zetten, samen met wat ze ermee willen bereiken: Een schone wereld voor iedereen!

Via onze website kan je verschillende verhalen doorgaan, zelf keuzes maken en erachter komen waarom die keuzes het beste zijn voor de omgeving. Zodra je je keuze hebt gemaakt kan je zien waarom het wel of niet zuinig is en hoe de EGD het aan zou pakken!
